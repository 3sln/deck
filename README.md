# @3sln/reel

A zero-config, Markdown-based component playground and documentation tool.

> **Note:** This is a work-in-progress project.

## Features

- **Vite Plugin:** A simple Vite plugin transforms a Markdown file into a full-featured development server.
- **`<reel-demo>`:** A custom element for embedding live, hot-reloading component demos directly in your documentation.
- **Reactive Controls:** A powerful `demoDriver` API for creating reactive properties and custom UI panels for your demos.
